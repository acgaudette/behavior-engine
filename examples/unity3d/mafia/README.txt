Setup:

  1. Create a blank scene
  2. Add an empty object
  3. Add two components to the empty: 'ComponentManager' and 'Mafia'
  4. Tick the "Compile With Debug Labeling" checkbox
  5. Create a new prefab, with (1) a MafiaUniverseComponent and (2) a Trigger
  6. Drag the prefab into the ComponentManager's "Default Universe" field
  7. Open the console
  8. Press play
  9. Read the output in data/log.txt
